## For git initialization
```
git init
```

## To add all files
```
git add . 
```

## To check status of files(modified or staged)
```
git status
```
## To commit changes
```
git commit -m "message"
```
## To delete file
```
git rm filename
```
## Viewing the commit history
```
git log
```
## Undoing things
```
git commit --amend
```
